Manual barcode scanner DBS HC-3208 for Gedemin

  1. Enter setup
  1. Set custom prefix
  1. Add STX non-printable ASCII control characters ('0' & '2')
  1. Save
  1. Enable add custom prefixes
  1. Set suffix
  1. Add STX non-printable ASCII control characters ('0' & '3')
  1. Save
  1. Enable suffix
  1. Enable invisible character output
  1. Enable output CTRL key combination
  1. Exit setup
  1. !!! Restore ALL Factory Defaults !!!
